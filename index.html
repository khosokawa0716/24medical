ABOUT

私たち24MEDICALは地域医療を２４時間３６５日支えます。
現在、超高齢化社会の日本において在宅医療のニーズは高まっています。
地域の薬局は２４時間３６５日対応を求められますが、マンパワー不足しています。
他の薬局と連携し在宅医用の質を向上し、『全ての人々が最後の場所を選べる』環境を目指します。

Layout

ページを構成するヘッダーやメインのコンテンツエリア、サイドバーやフッターといったプロジェクト共通のコンテナーブロックのスタイルを定義します。

Object

OOCSSのコンセプトを元に、プロジェクトにおける繰り返されるビジュアルパターンをすべてObjectと定義します。FLOCSSでのObjectは、さらに次の3つのレイヤーに分けられます。

Object

  • Component 再利用可能なパーツです。プレフィックスに「c-」をつけます（例: .c-btn、.c-btn_primary）
  • Project omponentにするほどでもないパーツです。プレフィックスに「p-」をつけます（例: .p-article、p.article__title）
  • Utility 汎用クラスで単一のスタイルを持っています。プレフィックスに「u-」をつけます（例: .u-clearfix、.u-block）
